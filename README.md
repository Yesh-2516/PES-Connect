PES Connect – Next-Gen Campus App

PES Connect is a full-stack MERN web application designed to centralize campus activities such as events, club recruitments, and lost & 
found services into a single platform. It provides students with an organized and accessible interface to stay updated, while enabling clubs
and admins to post and manage announcements efficiently. Built using React, Node.js, Express, and MongoDB, the application ensures seamless
data flow, real-time updates, and a responsive user experience
